## 💡 Key Design Decisions

### 1. **No Monetization Yet**
*Why*: Build network effects first
- Free for all users
- No commission on anything
- Establishes goodwill
- Allows viral growth
- Can monetize in Phase 2

### 2. **Expert Verification First**
*Why*: Trust is the foundation
- Every expert manually verified
- Admin approval queue
- Verified badge shown
- Parents can trust the platform
- Prevents fake/spam experts

### 3. **Content as Competitive Advantage**
*Why*: Articles + videos differentiate us
- 50+ curated articles
- 20+ expert videos
- Guides per condition
- Becomes SEO moat
- Compounds over time

### 4. **Forums for Engagement**
*Why*: Parents need peer support
- Discussions by condition
- Experience sharing
- Q&A format
- Moderation system
- Low feature set (fast to build)

### 5. **Search for Discovery**
*Why*: Parents are lost and searching
- Search experts by country + specialty
- Browse conditions with resources
- Find nearby services
- Simple, powerful discovery

## ⚠️ Critical Success Factors

### 1. **Maintain Scope**
- NO donations in MVP
- NO appointments in MVP
- NO progress tracking in MVP
- Follow the plan strictly
- Track feature requests for Phase 2

### 2. **Expert Verification**
- Every expert manually reviewed
- Quality > speed for this
- Verified badge is your moat
- Worth 2-3 extra days of work

### 3. **Community Moderation**
- Forum moderation is critical
- Hire community manager early (Week 4+)
- Set clear community guidelines
- Enforce respectfully

### 4. **Content Quality**
- Articles must be well-written
- Videos must be professional
- Hire freelance writers if needed
- This is your competitive advantage

### 5. **Mobile First**
- 60%+ traffic will be mobile
- Test on real devices
- Touch interactions
- Performance on slow connections

### 6. **Performance**
- Page load < 3 seconds
- API response < 500ms
- Optimize early, not later
- Use Vercel's analytics
