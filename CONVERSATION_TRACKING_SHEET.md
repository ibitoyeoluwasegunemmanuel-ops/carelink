# Conversation Tracking Sheet

**Purpose**: Log every customer conversation and outcome. This is the source of truth for real traction.

**Update**: After every conversation (same day if possible)

---

## EXPERT CONVERSATIONS

Create Google Sheet with these columns:

| Name | Profession | Location | Date Contacted | Method | Reply? | Date Replied | Interested | Meeting Scheduled | Date Scheduled | Notes | Status |
|------|-----------|----------|----------------|--------|--------|--------------|-----------|-------------------|----------------|-------|--------|
| [Expert Name] | Speech Therapist | Lagos | 2026-06-05 | Email | Yes | 2026-06-06 | Yes | Yes | 2026-06-10 | Enthusiastic, 50+ families/month | Conversation Scheduled |
| | | | | | | | | | | | |

**Column Definitions**:

- **Name**: Full name of professional
- **Profession**: Speech Therapist / OT / Psychologist
- **Location**: City
- **Date Contacted**: When you first reached out
- **Method**: Email / Phone / WhatsApp / LinkedIn
- **Reply?**: Yes / No / In Progress
- **Date Replied**: When they responded
- **Interested**: Yes / No / Maybe / Asked Questions
- **Meeting Scheduled**: Yes / No
- **Date Scheduled**: When meeting is set for
- **Notes**: Key quotes, pain points mentioned, objections
- **Status**: Contacted / No Reply / Declined / Interested / Meeting Scheduled / Meeting Completed / Verified Expert

**Tracking Rules**:
- Update SAME DAY after contact
- "No reply" = 7 days passed, no response
- "Maybe" = They asked follow-up questions
- "Interested" = Explicitly said they'd join or asked to sign up
- "Verified Expert" = They've completed the verification process

## What Counts as "Traction"

✅ **COUNTS** (Real signals):
- Professional responded to initial contact
- Professional expressed interest in joining
- Professional asked questions about the platform
- Parent filled out waitlist form
- Parent asked to join community
- NGO scheduled a meeting
- Partnership contact engaged in conversation

❌ **DOES NOT COUNT** (Vanity metrics):
- Email sent (no reply)
- Message delivered (no response)
- Form viewed (not submitted)
- Landing page visited (no action)
- Someone politely said "maybe"

---

## Conversation Quality Indicators

**Strong interest** (count this):
- Asked specific questions about how it works
- Mentioned a specific pain point
- Offered to help/recruit others
- Agreed to a meeting
- Asked when it launches

**Weak interest** (don't count):
- "Sounds interesting"
- "Maybe, let me think about it"
- "Sounds good, good luck"
- "I don't have time"
- Delayed response after follow-up

---

## Template for New Conversations

**Before calling/emailing**:
1. Add person to tracking sheet
2. Note method of contact
3. Note exact time contacted

**After conversation**:
1. Update "Reply?" column
2. Update "Interested?" column
3. Log key quotes in Notes
4. If meeting scheduled, set date
5. If objection, document it
6. Next follow-up date (if any)
