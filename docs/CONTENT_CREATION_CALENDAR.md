## Editorial Standards & Checklist

### Writing Standards

Every article must be:

```
Accurate ✓
├─ All facts checked against credible sources
├─ Medical claims evidence-based
├─ No misleading information
├─ "Not medical advice" disclaimer where needed

Actionable ✓
├─ Includes practical strategies
├─ Step-by-step guidance
├─ Real-world examples
├─ "What to do next" section

Empathetic ✓
├─ Tone warm and supportive
├─ Non-judgmental language
├─ Acknowledges parent challenges
├─ Celebratory of wins

Clear ✓
├─ Written for parent audience (not medical jargon)
├─ Short paragraphs (2-3 sentences max)
├─ Headers for scannability
├─ Key takeaways highlighted

Complete ✓
├─ Covers main topic thoroughly
├─ Addresses common questions
├─ Includes resources/links
├─ Provides next steps

Accessible ✓
├─ Large, readable font
├─ Good contrast
├─ Mobile responsive
├─ Alternative text for images
```

### Article Structure Template

```
Title
├─ Clear, benefit-oriented
├─ Include target keyword
└─ Keep under 70 characters

Meta Description (158 characters max)
├─ Summarize key benefit
├─ Include keyword
└─ Make compelling for search results

Introduction (100-200 words)
├─ Hook: Why this matters
├─ Context: What we're covering
├─ Roadmap: What parent will learn
└─ CTA: "Keep reading to learn..."

Body Sections (800-1,000 words)
├─ Section 1: Background/Understanding
├─ Section 2: Key strategies/options
├─ Section 3: Practical implementation
├─ Section 4: Common challenges & solutions
└─ Each section: 200-250 words with headers

Expert Insight (100-200 words)
├─ Quote from or perspective from expert
├─ Professional tip/recommendation
├─ Adds credibility
└─ Breaks up text

Action Steps (150-200 words)
├─ Clear, numbered steps
├─ Practical & doable
├─ Realistic timeline
└─ Encouragement

Resources (100-150 words)
├─ Where to learn more
├─ Organizations/links
├─ Books/tools mentioned
└─ Next professional steps

Disclaimer (if medical content)
├─ "This is not medical advice"
├─ "Consult with healthcare provider"
└─ "Information for educational purposes"

Total: 1,200-1,500 words
```

### Expert Review Checklist

Expert reviewers should check:

```
Accuracy
☐ All medical facts are correct
☐ Terminology is used properly
☐ Conditions described accurately
☐ Treatment options are current/evidence-based
☐ Statistics/research cited correctly
☐ No overstated claims

Completeness
☐ Main topic covered thoroughly
☐ Common questions addressed
☐ Different perspectives included
☐ Enough detail for parent understanding
☐ No critical information missing

Tone
☐ Appropriately empathetic
☐ Non-judgmental
☐ Encouraging without being unrealistic
☐ Professional but accessible
☐ No medical jargon without explanation

Safety
☐ No harmful recommendations
☐ No alternative "medicine" promoted without evidence
☐ Proper disclaimers included
☐ Clear when to seek professional help
☐ Emergency warning signs mentioned if relevant

Quality
☐ Well-written and clear
☐ Logical flow
☐ Good use of examples
☐ Actionable advice
☐ Proper citations/sources
```

### Copy Editor Checklist

Copy editors should check:

```
Grammar & Spelling
☐ No grammatical errors
☐ Spelling correct throughout
☐ Consistent capitalization
☐ Proper punctuation
☐ British or American English (consistent)

Style & Clarity
☐ Consistent tone throughout
☐ Clear sentence structure
☐ No jargon without explanation
☐ Active voice preferred over passive
☐ Short sentences preferred

Formatting
☐ Headers properly formatted
☐ Lists use consistent formatting
☐ Bold/italics used appropriately
☐ Line breaks for readability
☐ Consistent spacing

Consistency
☐ Terms used consistently
☐ Style guide followed
☐ Terminology matches across articles
☐ Cross-references correct
☐ Links functional

Accessibility
☐ Alternative text on all images
☐ Links descriptive (not "click here")
☐ Proper heading hierarchy
☐ Sufficient color contrast
☐ Large enough font size
```

## Risk Management

### Risk 1: Writers Miss Deadlines

```
Mitigation:
├─ Assign fewer articles to each writer if needed
├─ Start earlier in week
├─ Use templates to speed up writing
├─ Plan for 20% buffer time
└─ Have backup writer on call
```

### Risk 2: Expert Review Bottleneck

```
Mitigation:
├─ Recruit 5 reviewers instead of 3
├─ Assign specific articles to avoid overlap
├─ Set clear review SLA (2-3 days max)
├─ Use staggered timing (don't send all at once)
└─ Prepare backup reviewer for urgent pieces
```

### Risk 3: Quality Issues Missed

```
Mitigation:
├─ Editor does second review before publishing
├─ Copy editor catches final issues
├─ Use fact-checking tools
├─ Expert review mandatory before publishing
└─ Publish first pieces Monday so fixes can be made
```

### Risk 4: Topic Redundancy

```
Mitigation:
├─ Editor ensures each article is unique
├─ Cross-check topics before assigning
├─ Ensure different angles on same topic
└─ Use outline review to catch duplication
```

### Risk 5: Not Enough Discussion Content

```
Mitigation:
├─ Discussion starters are short (can do in parallel)
├─ Editor writes many personally
├─ Use content calendar to prioritize
├─ Can create additional starters in week 7
└─ Quality over quantity (50 minimum, 60+ ideal)
```
