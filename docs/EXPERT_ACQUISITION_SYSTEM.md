### Step 3: Credential Verification

```

│   - Driver's license
│

│   - Medical license (if doctor)

├── Education Verification
│   - Degree certificate(s)
│   - Transcripts (optional)

    - Proof of current employment
    - Professional association membership (if applicable)
    - References (2 professional references)
```

**Verification Process**

```
Phase 1: Document Check (1-2 days)
├─ All documents received?
├─ Documents readable and complete?
├─ Information consistent across documents?
└─ Flag any issues

Phase 2: Background Check (2-3 days)
├─ Verify with professional boards
│  - Medical & Therapy licensing boards
│  - Psychology boards
│  - Teaching credential verification
├─ Check professional associations
├─ Verify current employment
└─ Check for disciplinary records

Phase 3: Reference Calls (2 days)
├─ Call professional references
├─ Verify employment history
├─ Confirm expertise and specialization
└─ Ask about reputation and reliability

Phase 4: Final Review (1 day)
├─ All checks passed?
├─ Expert can be trusted?
├─ Profile accurate and complete?
└─ Ready for approval
```

**Verification Checklist**


## Expert Verification Roles & Responsibilities

### Expert Verification Team (2 people)
- **Verification Manager**: Oversees process, makes decisions
- **Verification Specialist**: Conducts background checks, calls references

### Timeline Per Expert
- **Application**: Day 1
- **Document check**: Days 2-3
- **Background check**: Days 4-6
- **Reference calls**: Days 6-7
- **Approval**: Day 8
- **Total**: ~1 week per expert

### Parallel Processing
- Process 3-4 experts per week (if 2 people)
- 100 experts = 25 weeks at 1 per week
- **Need to process 2-3 per week to hit 100 by launch**

### Acceleration Strategies
```
1. Hire verification team of 3 instead of 2
   → Can process 6+ per week
   → Reduces timeline to 16 weeks

2. Use checklist-based verification
   → Faster, more consistent
   → Less subjective decisions

3. Automate document checks
   → Use OCR for ID verification
   → Use online license checkers
   → Reduces manual work by 30%

4. Batch reference calls
   → Call all references on same day
   → Reduces turnaround time

5. Prioritize by specialization
   → Process high-demand specialists first
   → Ensure good mix at launch
```
